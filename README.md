# Python Automation Testing – Login & Registration Page

This project automates testing of Login and Registration pages using
Python, Selenium WebDriver, and PyTest framework.

## Tech Stack
- Python
- Selenium
- PyTest
- ChromeDriver

## Features Tested
- Valid Login
- Invalid Login
- User Registration
- Field Validations

## Project Structure
- pages/ → Page Object Model classes
- tests/ → Test cases
- utils/ → Configuration
- drivers/ → Browser driver
- sample_web_pages/ → HTML pages for testing

## How to Run
1. Install dependencies
   pip install -r requirements.txt

2. Run tests
   pytest -v
